About This Application

Introduction

This application is a Simple CRUD (Create, Read, Update, Delete) Web Application built using HTML, CSS, JavaScript, and a backend technology like Node.
js with Express and MongoDB.
It allows users to manage and manipulate data efficiently through a user-friendly interface.

Objective

The main objective of this CRUD application is to demonstrate how data can be created, displayed, updated, and deleted in a structured manner.
It follows a step-by-step, professional approach with animations to enhance user experience.

Key Features

  • Create (C) – Users can add new records through an input form.
  • Read (R) – Users can view all stored records dynamically.
  • Update (U) – Users can edit existing records and update data.
  • Delete (D) – Users can remove unwanted records from the database.

Technology Stack

  • Front-end: HTML, CSS, JavaScript
  • Back-end: Node.js and Express.js
  • Database: MongoDB
  • Additional Enhancements: Animations and transitions
